# 🍕 Pizza Sales SQL Project

This project demonstrates SQL proficiency by analyzing pizza sales data.  
The repository includes both the **business questions** and their **SQL solutions**, organized by difficulty level.

---

- **Concepts:** Joins, Aggregations, Grouping, Window Functions

This project highlights:
- Data aggregation and reporting  
- Business insights generation  
- Query optimization and structured problem solving
